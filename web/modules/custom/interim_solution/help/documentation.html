Overview

This document explains how to work with the Interim Solution section.

How it works

You have to fill out this information in the Finding aid form, so the system searches for its assets in the Catalog:

Routes

Interim Solution Settings

It is on /admin/config/interim-solution/settings

If you check "Automatic Switch" option, you won't have to approve the Finding aids' Asset from Catalog to show them to the users.

You have to add an email address to get a notification when a Finding aid has assets in the Catalog. If this text field is empty, it won't send any email.

Interim Solution Report

It can find on /admin/reports/interim-solution

It gives some information about the Interim Solution.

Finding Aids

You can find it on /admin/content/finding_aids

Here you can delete the Medias associated with the Finding aids that have assets in the catalog, and they were approved by a content editor.

The red color is for the contents that you cannot delete because they do not meet the conditions to that.

All administrators can access these routes, but you can give permission to other users to access them.

Queues

There are two background processes in Drupal which are checking if there is Finding aids with assets in the Catalog. The first one is for newly Finding aids created, it searches if there are assets in the Catalog, and the second one is responsible for finding updates that info when they already have assets in there.
